인생책장

프로젝트 설명

고령층 우울증의 유병률이 증가하고 있는 상황에서, 충분한 사회적 지원 시스템의 부재가 문제 해결을 어렵게 하고 있다. 자서전 작성이 노인 우울증 감소에 효과적임을 확인하였으므로, 이를 기반으로 (주)이분의일코리아와 협력하여, 생성형 AI를 활용한 구술 자서전 애플리케이션 개발을 추진하고자 했다.

기술 스택

Flutter Spring AWS Azure Prompt Flow

기본 정보

참여인원
4명 (FE 3, BE 1)
기간
2024.03 ~ 2025.02
관련 링크

수행 역할(FE)

  1. 온보딩 페이지 개발
    • 사용자의 상세 정보를 입력받는 기능 구현
    • 입력된 정보를 바탕으로 생성형 AI를 활용해 사용자 맞춤형 자서전 주제를 생성하는 기능 구현
  2. 교정/교열 기능 개발
    • 생성형 AI가 수정한 내용을 초록색 글씨, 사용자가 입력한 원래 내용을 빨간색 글씨로 표시하여, 사용자가 AI의 수정을 원하지 않을 경우 원래 내용을 유지할 수 있도록 기능 구현
  3. Azure 프롬프트 플로우 활용
    • 챕터 생성 과정에서 프롬프트 플로우를 적용하여, p-value가 3점 이상인 결과물만 사용하여 사용자의 의도와 다르게 내용이 벗어나지 않도록 설정

활동 사진

봄멍

프로젝트 설명

유기견 입양에 대한 관심이 점점 높아지고 있지만, 유기견과 반려인이 매칭되는 확률은 여전히 낮다. 이에 매칭확률을 높이고자 “포인핸드”에서는 강아지의 성격, 각종 좋아하는 것 등 다양한 정보를 제공해주고자 노력하지만, 입양을 고민하는 이들에겐 아직 어렵게 다가오고 있다. 이를 해결하기 위해 생성형 AI를 활용한 “유기견의 페르소나”를 만들어, 유기견과 직접 대화하는 듯한 기능을 통해 유기견과 예비 반려인이 매칭할 수 있는 여지를 높이고자 했다. 최종적으로는 유기견을 관심있게 바라볼 수 있는 새로운 입양 시스템을 만들고자 했다.

응급실 혼잡도 알리미

프로젝트 설명

일반 병원이 진료를 보지 않는 시각에는, 필연적으로 응급실에 갈 수 밖에 없다. 하지만 환자 입장에서는 현재 어떤 응급실이 혼잡한지 알 수 없다. 또한 소아과 담당 주치의가 없는 응급실에 위급한 소아 환자가 가게 되면, 진료를 봐주지 않는 문제가 발생하여 소위 말하는 “응급실 뺑뺑이” 현상이 일어나게 된다. 이를 해결하기 위해, 응급실 혼잡도와 소아 진료가 가능한 응급실을 알려주는 웹/앱을 만들고자 했다.

기술 스택

React Spring Docker

기본 정보

참여인원
4명 (FE 1, BE 3)
기간
2023.06 ~ 2023.08
관련 링크

수행 역할(FE, PM)

  1. 기획 역할 수행
    • 응급의료정보제공 이라는 앱에서도 보여주지 않는 “응급실 혼잡도”를 앱의 메인 정보로 설정하고, 프로젝트 기획을 시작함. 이에 프로젝트 개요서, 플로우 차트, IA(정보 구조도), 와이어 프레임, 디자인과 같은 작업을 전담하여 프로젝트를 구축함
  2. React를 중심으로 FE 역할 수행
    • “지도 기준 응급실 찾기” 기능의 스크린에서 사용자의 웹/앱 화면의 크기에 따라 응급실 혼잡도 아이콘이 유동적으로 보일 수 있도록 함
    • Daum 지도 api를 활용하여 지도, 길찾기 등과 같은 위치 활용 기능에서 사용자의 현재 위치 탐색, 응급실까지의 길찾기를 수행할 수 있도록 함
